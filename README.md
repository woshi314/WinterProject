## 第一次作业提交
pocket_monster系统

### 实现了基本的要求功能

### 特色
#### json文件存储怪兽图鉴配置
支持反序列化json文件
方便修改怪兽种类的各项配置,如属性和描述
#### 接入openai的api,支持ai自动生成故事内容
将所有怪兽的信息和设定好的提示词,以及用户输入的提示词合并
返回的内容作为故事内容
因此可以结合背景,联动其他怪兽和故事
